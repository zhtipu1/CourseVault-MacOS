CourseVault

Offline Course Manager

v1.3.0 Python · Flask · pywebview · SQLite

CourseVault is a fully offline desktop application for organizing and watching your video courses. Import course folders or build courses manually, track your progress — all stored locally. No accounts, no cloud, no tracking.

Features

  • Automatic folder scanning with multi-level module detection
  • Manual course builder for full control over structure
  • Progress tracking with resume playback
  • Timestamped notes and bookmarks, exportable to Markdown
  • Subtitle support (.srt / .vtt)
  • Wide format support — MKV, AVI, TS and more via FFmpeg
